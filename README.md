# Zapatilla Rosa 👠

Zapatilla Rosa es una página web de venta de zapatos para mujer desarrollada con HTML, CSS, Tailwind CSS y JavaScript. El proyecto fue creado con el objetivo de ofrecer una experiencia de compra moderna, atractiva y responsiva para diferentes dispositivos.

## Características

* Diseño adaptable para computadoras, tablets y celulares.
* Menú de navegación responsive.
* Carrusel interactivo con promociones y descuentos.
* Catálogo de productos cargado dinámicamente desde una API.
* Visualización de precios y descuentos.
* Botón de compra directa mediante WhatsApp.
* Sistema de favoritos para marcar productos.
* Efecto Zoom Card en las imágenes de los zapatos.
* Descripción emergente al pasar el cursor sobre los productos.
* Sistema de reseñas con calificación por estrellas.
* Almacenamiento local de comentarios mediante Local Storage.

## Tecnologías utilizadas

* HTML5
* CSS3
* Tailwind CSS
* JavaScript

## Objetivo del proyecto

Desarrollar una tienda virtual funcional que combine diseño moderno, interactividad y una experiencia de usuario intuitiva, aplicando conceptos de desarrollo web frontend.

## Mejoras Futuras

Este proyecto puede seguir evolucionando con nuevas funcionalidades para mejorar la experiencia de los usuarios:

* Implementar un carrito de compras para agregar varios productos antes de realizar el pedido.
* Agregar filtros por categoría, talla, color y precio.
* Incorporar una barra de búsqueda para encontrar productos más rápidamente.
* Crear un sistema de inicio de sesión y registro de usuarios.
* Permitir subir imágenes en las reseñas de los productos.
* Mostrar la calificación promedio de cada producto basada en las reseñas de los usuarios.
* Integrar una base de datos para almacenar productos, usuarios y comentarios de forma permanente.
* Añadir un sistema de pagos en línea mediante pasarelas de pago.
* Implementar un modo oscuro para mejorar la accesibilidad y la experiencia visual.
* Agregar una sección de productos recomendados o más vendidos.
* Incluir animaciones más avanzadas para mejorar la interacción con los productos.
* Optimizar el rendimiento de la página para cargas más rápidas.
* Incorporar notificaciones para informar sobre promociones y nuevos productos.
* Añadir una galería con varias imágenes por producto y función de zoom avanzado.
